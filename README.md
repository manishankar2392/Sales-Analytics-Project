# 📊 Sales Analytics Dashboard using Microsoft Excel

An interactive **Sales Analytics Dashboard** built in **Microsoft Excel** using Pivot Tables, Pivot Charts, Slicers, Timelines, and Excel Dashboard techniques. This project helps analyze sales performance, product trends, salesperson performance, and regional sales with an easy-to-use interface.

---

## 📌 Project Overview

This dashboard provides business insights through interactive visualizations and allows users to filter data dynamically by:

- 📅 Date
- 📍 Place
- 📦 Product Category

It enables quick analysis of sales performance, best-selling products, top-performing salespersons, and sales trends.
